关于向素

向素是一个简单、快速建站平台，用户选择喜欢的网站模板后，不需要任何设计和代码基础就可以快速搭建一个属于自己的网站。用户只需要把展示的内容上传到网站不同的模块中，通过使用可视化控件快速的自定义网站风格、样式，真正做到零设计，零代码。同时为用户提供后台托管和维护等服务，极大节省了用户建立个人网站的效率和时间。

我们致力于为艺术从业者和自由职业者，如设计师、艺术家、摄影师等用户提供优质的个人网站模板，使用模板式建站的方式，在不依赖设计师和程序员的情况下，帮助用户可以更低成本，更快，更便捷的完成建立个人网站的工作，用户可以更好的自我推广，并建立自己的品牌和形象。

在历时半年的策划、设计、开发和打磨之后，四月下旬开始内测，将于五月份以最简单的方式和大家见面。暂时提供一个网站模板，首页、画廊、博客这三个最基本的功能，以及部分可视化修改控件来调整网站风格。在未来，我们会为用户提供更多优质的个人网站模板，添加更多实用，有效的功能，帮助用户做出更满意的网站。

展览合作

毕业展是我们联合高校策划的一个高校艺术类毕业生线上展览的合作，通过毕业展的活动，为高校毕业生提供线上展览的活动，让跟多的学校，更多的学生作品得到曝光的机会，同时为企业找到更合适的毕业生提供更便捷的途径。毕业生也可以把自己的作品以链接的形式分享给用人单位，让毕业季线上展览成为学生步入社会的延伸。

诚挚的邀请并欢迎高校加入毕业展，让更多的人足不出户就能看到高校学生的作品。
